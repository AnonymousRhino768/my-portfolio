My Projects

First React Project

For this project, I made a list of politicians whom might be running in the 2024 presidential election. The site has the capability to filter based on the parties of the candidates, and can sort the candidates by number of votes. Lastly, users can choose to follow different politicians and see the total number of votes the politicians they have followed have. This project was my first time using React and it helped me learn and understand the language and its components. If I were to do this project again I would add more CSS and styling to the site to make it more visually appealing.
